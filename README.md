# DriftAware-HeartDisease-PredictionModel-EBM
Drift-aware Explainable Boosting Machine (EBM) for interpretable heart disease prediction with cross-population validation, feature-level explainability, Population Stability Index (PSI)-based drift detection, and simulated clinical drift analysis for reliable AI deployment in healthcare.

      Note: This repository contains the research implementation accompanying the published paper. It is intended for research and educational purposes. Users are encouraged to validate results within their own experimental settings.

# Publication

This work was presented at the IEEE 12th International Conference on Bio Signals, Images, and Instrumentation (ICBSII 2026).
      DOI: 10.1109/ICBSII69710.2026.11479260

# Overview

Artificial Intelligence (AI) models used in healthcare often suffer from concept drift, where predictive performance degrades due to changes in patient populations, clinical practices, or data distributions over time. This repository presents a Drift-aware Explainable Boosting Machine (EBM) framework for interpretable heart disease prediction that combines transparent feature-level explanations with Population Stability Index (PSI)-based drift detection. The framework is evaluated using in-distribution testing, cross-population validation, and simulated clinical drift analysis to assess its reliability for real-world healthcare deployment.
