# Dataset

This project uses the UCI Heart Disease dataset.

# Download

https://archive.ics.uci.edu/dataset/45/heart+disease

# Instructions

1. Download the dataset.
2. Run `src/preprocessing.py`.
3. The cleaned dataset will be generated automatically.
4. The processed dataset will be saved to:

```
data/processed/heart_cleveland_clean.csv
```

# The processed dataset is not included in this repository.
